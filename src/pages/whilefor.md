While loop

```lua
local i = 0
local count = 0

while i <= 10 do
  count = count + 1
end

print("count is " .. count)
--count is 7
```

<br>For loop

```lua
count = 0
for i=1, 5 do
    count = count + 1
end
print("count is " .. count)
--count is 5
```
