Ruby provides common [mathematical functions](https://ruby-doc.org/core-2.6/Math.html) out of the box.

```ruby
x = 0.5
puts Math.acos x
# 1.0471975511965979
```

Full list with paremeter and return types:

```
Math::acos x -> Float
Math::acosh x -> Float
Math::asin x -> Float
Math::asinh x -> Float
Math::atan x -> Float
Math::atan2 y, x -> Float
Math::atanh x -> Float
Math::cbrt x -> Float
Math::cos x -> Float
Math::cosh x -> Float
Math::erf x -> Float
Math::erfc x -> Float
Math::exp x -> Float
Math::frexp x -> Float
Math::gamma x -> Float
Math::hypot x, y -> Float
Math::ldexp fraction, exponent -> [Float, Float]
Math::lgamma x -> [Float, -1 or 1]
Math::log *args
Math::log10 x -> Float
Math::log2 x -> Float
Math::sin x -> Float
Math::sinh x -> Float
Math::sqrt x -> Float
Math::tan x -> Float
Math::tanh x -> Float
```
