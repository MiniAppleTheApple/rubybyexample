You can try ruby [interactively](https://try.ruby-lang.org/).

```ruby
# This is a comment.
puts("Hello, World!")

# Parenthesis are optional

puts "Hello, World!"

=begin
    This is a
    multi-line comment.
=end
```

To run the code put it in `hello.rb` and use `ruby hello.rb`.
